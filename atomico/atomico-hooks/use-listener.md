# use-listener

### Module

```javascript
import { useListener } from "@atomico/hooks/use-listener";
```

### Syntax

```jsx
useListener(
    ref: Ref<Element>, 
    name: string, 
    handler: (event)=>void, 
    option?: boolean | AddEventListenerOptions 
);
```
