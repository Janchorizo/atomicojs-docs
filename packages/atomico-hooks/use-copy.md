# use-copy

### Module

```javascript
import { useCopy } from "@atomico/hooks/use-copy";
```

### Syntax

```javascript
const copy:()=>void = useCopy(ref);
```

Where:

1. `ref` : Node reference to copy
2. `copy`: Callback, executes the content copy event.
